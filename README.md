> A command-line based Contact Management System built in C — my first project as a BCA student. Features full CRUD operations with persistent file storage.

---

## Features
- ➕ Add new contacts (name, phone, email)
- 🗑️ Delete existing contacts
- 🔍 Search contacts by name
- 📋 Display all saved contacts
- 💾 Save contacts to a file
- 📂 Load contacts from a file

---

## How to Run
```bash
# Compile
gcc Contact-Management.c -o contacts

# Run
./Contact-Management
```
